# Graph Pathfinder

A cool little graph pathfinder built with React!

Currently supports two graph algorithms.
- Breadth First Search (BFS)
- Depth First Search (DFS)

More coming soon.

Try it out [here](https://ad-world.github.io/pathfinder).

## Stack
- React
- TypeScript
- Chakra UI
- ESLint
- Vite
- Github Pages
